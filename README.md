## 🌟 About
<p align="center">
  <strong>AlgoEngine</strong> is a next-generation algorithmic assessment and visualization platform designed to revolutionize how developers and students learn Data Structures & Algorithms. Combining interactive problem-solving, real-time visualizations, comprehensive quizzes, and intelligent analytics into one elegant ecosystem.
</p>

<table width="100%" cellspacing="0" cellpadding="10">
  <tr>
    <td width="50%" valign="top">
      <h3>💎 Why AlgoEngine?</h3>
      <ul>
        <li><strong>Visual Learning:</strong> Watch algorithms animate in real-time</li>
        <li><strong>Comprehensive Platform:</strong> DSA + CN + OS + DBMS + Aptitude</li>
        <li><strong>Smart Analytics:</strong> Track every metric that matters</li>
        <li><strong>Gamified Experience:</strong> Earn badges and climb the ranks</li>
      </ul>
    </td>
    <td width="50%" valign="top">
      <h3>🎯 Built For</h3>
      <ul>
        <li>Students preparing for technical interviews</li>
        <li>Developers mastering algorithmic thinking</li>
        <li>Educators teaching data structures</li>
        <li>Anyone passionate about coding excellence</li>
      </ul>
    </td>

📦 Step 1: Clone the Repository
<pre><code class="lang-bash">> git clone https://github.com/sohamrajput98/AlgorithmicEngine.git > cd AlgorithmicEngine </code></pre>

🔧 Step 2: Backend Setup
<table width="100%" cellspacing="0" cellpadding="10"> <tr> <td width="50%" valign="top"> <h4>📝 Manual Setup</h4> <pre><code class="lang-bash"># Install dependencies > pip install -r requirements.txt

Run migrations
alembic upgrade head

Start server
uvicorn app.main:app --reload </code></pre> </td> <td width="50%" valign="top"> <h4>⚡ Automated Scripts</h4>

<pre><code class="lang-bash">> cd backend/app/scripts

Development mode
> ./dev.sh

Production mode
> ./prod.sh

Run tests
> ./test.sh </code></pre>

</td>
</tr> </table>

🎨 Step 3 & 4: Frontend & Database
<table width="100%" cellspacing="0" cellpadding="10"> <tr> <td width="50%" valign="top"> <h4>🎨 Step 3: Frontend Setup</h4> <pre><code class="lang-bash">> cd frontend

Install dependencies
npm install

Start Vite dev server
npm run dev </code></pre> </td> <td width="50%" valign="top"> <h4>🗄️ Step 4: Database Initialization</h4>

<pre><code class="lang-bash">> cd algorithmicengine/scripts

Create databases
> ./create_databases.sh

Backup database
> ./backup.sh </code></pre>This-is-a-placeholder-for-the-second-column-content

</td>
</tr> </table>

⚙️ Step 5: Environment Configuration
Create a .env file in the backend directory:

<pre><code class="lang-ini">DATABASE_URL=mysql://user:password@localhost/algoengine SECRET_KEY=your-secret-key-here ALGORITHM=HS256 ACCESS_TOKEN_EXPIRE_MINUTES=30 </code></pre>
